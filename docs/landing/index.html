核心特色

為上課而生，不只是另一個錄音 App

每一個功能都為了一個目標：讓英文授課變得好跟得上、好複習、好整理。

即時英文轉錄

本地 Whisper 推論，完全離線可用。從 Tiny 到 Large-v3 Turbo 依硬體自選，~2 秒延遲的滾動式字幕。

兩階段翻譯

即時粗翻秒回，背景 LLM 精修再自動覆寫上去。字幕一開始就有，看完整課後全部升級成自然流暢版本。

帶著你的 AI 來

Copilot Pro、ChatGPT Plus OAuth、OpenAI / Anthropic / Gemini API key 都支援。用你已經付過錢的訂閱，不用再開一份。

PDF 投影片同步

上傳講義，字幕自動對齊到你正在講的那張投影片。複習時點一下任一頁，直接跳到那段錄音。

RAG 課程問答

下課後和你的筆記對話。投影片 + 錄音都切塊算 embedding（本地 Candle 跑），AI 答題自動引用出自哪張投影片、哪段時間。

GPU 加速

Windows CUDA / macOS Metal 皆支援。RTX 4060 Ti 上 Whisper 推論 ~20× realtime，一小時的課 3 分鐘轉完。
